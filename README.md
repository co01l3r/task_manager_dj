# Task Manager
![GitHub last commit](https://img.shields.io/github/last-commit/co01l3r/task_manager_dj)

simple task manager

#### how to use:
```python
$ virtualenv <env_name>
$ source <env_name>/bin/activate
(<env_name>) $ pip install django

from project folder:

(<env_name>) $ python3 manage.py runserver
```
